Smart Contactless Health Monitoring Chair — Simulation

This project simulates a contactless health monitoring chair using Streamlit. It models BCG + respiration signals and an ANN model for BP estimation.

How to Deploy on Streamlit Cloud:
1. Go to https://github.com → create a new public repository.
2. Upload these files to the repo:
   - streamlit_app.py
   - requirements.txt
   - README.txt (optional)
3. Go to https://share.streamlit.io, log in with GitHub, click "New app".
4. Choose your repository and set Main file path to streamlit_app.py
5. Click Deploy. Wait a few minutes for the app to build and go live.

How to run locally:
pip install -r requirements.txt
streamlit run streamlit_app.py
